# microbot-py-auto-credential
Automatically adds credentials to microbot launcher.

1) Pull credentials from txt file and seperate into email/password
2) Find Microbot add account button and click it
3) Wait for browser pop up then fill out the username/password
4) When asking for email confirmation open new browser
5) Login to the email account on the new browser and find the first email code
6) Take the email code and fill it into the other browser
7) Update credentials file that its completed, repeat
